Типи даних script-typedata.js

Типи даних JS

  1. Undefinate - невизначено
  2. Null - немає значення тобто 0
  3. Boolean - 1 (TRUE)> 0(FALSE)
  4. Number - число
  5. BigInt - велике число (виділяється більше ОЗП)
  6. String - рядок
  7. Object - об'єкт, що містить різні властивості , а отже включає в себе різні типи
  8. Symbol - унікальний ID об'єкта
  9. Function - об'єкт (що приймає та повертає дані, значення різних типів)
  10. NaN

Функції перетворення типів даних

Існують варіанти автоматичного перетворення типу даних у JS (наприклад під час ділення рядково типу на числовий , де рядковий буде перетворено в числовий тип. Але у випадках неможливості обчислення "тип" буде NaN)

  1. ... = String();
  2. ... = Number();
  3. ... = Boolean(NaN, 0 , undefinate)=> FALSE;
